# Features

## Frontend Features

* React-based user interface
* Multiple pages using React Router
* Todo List page
* Todo Details page
* Add Todo
* Edit Todo
* Delete Todo
* Fetch data using Axios
* Responsive layout

---

## Backend Features

* Express.js REST APIs
* MVC Architecture
* CRUD Operations
* JSON File Storage
* Error Handling
* CORS Enabled

---

## APIs

* GET /todos
* GET /todos/:id
* POST /todos
* PUT /todos/:id
* DELETE /todos/:id

---

## Project Highlights

* Full Stack Application
